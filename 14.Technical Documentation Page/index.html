Introduction

HTML (HyperText Markup Language) is the backbone of web development. It is used to create the structure of web pages. HTML works alongside CSS for styling and JavaScript for dynamic behavior.

HTML documents consist of elements, which are enclosed in angle brackets. These elements define different parts of a webpage, such as headings, paragraphs, lists, links, images, and more.

HTML files use the .html extension and are interpreted by web browsers to render web pages.

Basic HTML structure:

                <!DOCTYPE html>
                <html>
                <head>
                    <title>My First Page</title>
                </head>
                <body>
                    <p>Hello, World!</p>
                </body>
                </html>

HTML is not a programming language but a markup language used to structure content. It provides a way to create a web page's skeleton, which can later be styled and made interactive.

HTML has evolved significantly, with HTML5 being the latest version, offering new elements like <section>, <article>, and <nav>, improving semantic meaning and accessibility.

Developers use HTML in combination with CSS and JavaScript to create responsive, interactive websites.

Browsers like Chrome, Firefox, and Edge interpret HTML to display web content.

Understanding HTML is essential for web development, forming the base for building modern websites.

Elements

HTML elements are the fundamental building blocks of web pages. Each element consists of a start tag, content, and an end tag.

Example of a paragraph element:

<p>This is a paragraph.</p>

Some elements, like the <br> tag, are self-closing and do not require an end tag.

Common elements:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor (links)
  • <img> - Image
  • <ul> and <ol> - Lists

Each element serves a specific purpose in structuring a webpage.

For example, the <div> element acts as a container to group other elements together, which helps in styling and layout design.

HTML5 introduced new semantic elements such as <header>, <footer>, <section>, and <article>, making it easier to structure pages meaningfully.

Attributes

Attributes provide additional information about an HTML element. They are always included in the opening tag.

Example:

                <a href="https://example.com" target="_blank">Visit Example</a>

Common attributes:

  • href - Specifies the URL of a link.
  • src - Defines the source file of an image.
  • alt - Provides alternative text for images.
  • id - Assigns a unique identifier to an element.
  • class - Allows CSS styling of multiple elements.

Attributes improve accessibility and interactivity of webpages.

For example, the alt attribute in an image provides a textual description, useful for screen readers and cases where the image fails to load.

The title attribute provides a tooltip when the user hovers over an element.

Headings

HTML headings are used to define titles and subtitles in a webpage. There are six levels of headings, from <h1> (most important) to <h6> (least important).

Headings help structure content and improve SEO by making it easier for search engines to understand the hierarchy of information.

                <h1>Main Heading</h1>
                <h2>Subheading</h2>
                <h3>Section Heading</h3>

For accessibility, it's recommended to use heading levels correctly and not skip levels arbitrarily.

Paragraphs

The <p> element defines a paragraph of text.

Paragraphs help in separating blocks of text to enhance readability.

                <p>This is a paragraph.</p>

Browsers automatically add space before and after paragraphs to improve text flow.

Text Formatting

HTML provides various elements to format text:

  • <b> - Bold text
  • <i> - Italic text
  • <u> - Underlined text
  • <strong> - Important text
  • <em> - Emphasized text
                <b>Bold Text</b>
                <i>Italic Text</i>
                <u>Underlined Text</u>
                <strong>Strong Text</strong>
                <em>Emphasized Text</em>

The <strong> and <em> tags are useful for accessibility and screen readers.

Links

Links are created using the <a> tag.

                <a href="https://example.com" target="_blank">Visit Example</a>

The target="_blank" attribute opens the link in a new tab.

Links can also be styled using CSS to change color, add hover effects, and more.

Comments

Comments are used to add notes in the code that are not displayed on the webpage.

                <!-- This is a comment -->

Comments are useful for explaining code and making it easier to maintain.

Images

Images are added using the <img> tag.

                <img src="image.jpg" alt="Description of image" width="300">

The alt attribute provides alternative text for accessibility and SEO.

Images can also be styled using CSS for size, alignment, and effects.

Lists

There are two types of lists in HTML:

  • Ordered List (<ol>) - Numbered items
  • Unordered List (<ul>) - Bulleted items
                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                </ul>
                
                <ol>
                    <li>First</li>
                    <li>Second</li>
                </ol>

Lists are helpful for organizing content clearly.

Tables

Tables are created using the <table> tag.

                <table border="1">
                    <tr>
                        <th>Name</th>
                        <th>Age</th>
                    </tr>
                    <tr>
                        <td>Alice</td>
                        <td>25</td>
                    </tr>
                </table>

Tables are used for displaying structured data.

Forms

Forms allow user input via various input elements like text fields, checkboxes, and buttons.

Example:

                <form>
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name" required>
                    <input type="submit" value="Submit">
                </form>

Common form elements:

  • <input> - Used for various input types (text, email, password).
  • <textarea> - Multi-line text input.
  • <select> - Dropdown menu.
  • <button> - Clickable button.

Forms are used for login, signup, surveys, and more.

Each input field requires a name attribute to be identified when submitting data.

Form validation is an essential feature that ensures users enter correct data before submission. Attributes like required, minlength, and maxlength help in validation.

Modern forms use JavaScript and backend integration to process user data efficiently.
